# DEVOPS [DO-001]: Setup Docker Compose file for local development

## Description
Define services for PostgreSQL database, Redis instance, and Soroban RPC quickstart node.

## Files to Edit/Create
- `docker-compose.yml` at project root

## Acceptance Criteria
- `docker-compose up` launches all 3 backend dependencies.

## Guidance / Hints
Use official Soroban quickstart images configured for local sandbox.
